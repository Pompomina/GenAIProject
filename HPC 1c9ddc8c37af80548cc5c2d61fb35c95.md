**Singularity:**

A **Singularity Image** (often `.sif` file) is a single-file container image that includes:

- An operating system layer (e.g., Ubuntu).
- Libraries, drivers, or tools (e.g., CUDA, cuDNN for GPU computing).
- Application software (e.g., Python, conda, R, etc.).

A **Singularity image** (`.sif`) is typically read-only, so you often use an **overlay** file (`.ext3`) to have writable space.

- You may want to install **extra packages** (e.g., Python libraries with `pip`, or system packages with `apt`, or custom software). Since you cannot modify the original `.sif` file, you need a **writable layer** where these new files can go.

What does “**Mount**” mean:

- When you mount a filesystem (like an overlay file) onto a directory, you’re telling the system: “Treat this directory path as if it were the root of that filesystem.”
- `singularity exec --overlay /path/to/overlay.ext3 …`
    
    **`--overlay`** tells Singularity to layer the specified filesystem (the `.ext3` file) on top of the container’s read-only image. Now, when you install or create new files within certain directories in the container, they actually go into that overlay file.
    
- `singularity exec --bind /some/host/dir …`
    
    **`--bind`** (bind mounting) tells Singularity to mount a host directory (e.g., `/scratch/[username]/`) inside the container, so that the container sees it as if it’s part of its own filesystem tree.
    

How to setup for Singularity

1. request for CPU resources
    - `srun --account=ds_ga_1012-2025sp --partition=interactive --time=**04:00:00** --pty /bin/bash`
2. Copy over the empty filesystem image
    - `cd /scratch/js11052`
    - `scp greene-dtn:/scratch/work/public/overlay-fs-ext3/overlay-25GB-500K.ext3.gz .`
3. Unzip the ext3 filesystem
    - `gunzip -v ./overlay-25GB-500K.ext3.gz`
4. Choose and copy a Singularity image to GCP
    - `scp -rp greene-dtn:/scratch/work/public/singularity/cuda11.4.2-cudnn8.2.4-devel-ubuntu20.04.3.sif .`
5. Launch the Singularity container
    - `singularity exec --nv \
    --bind /scratch/bl1986/ \
    --overlay /scratch/bl1986/overlay-25GB-500K.ext3:rw \
    /scratch/bl1986/cuda11.4.2-cudnn8.2.4-devel-ubuntu20.04.3.sif \
    /bin/bash`
